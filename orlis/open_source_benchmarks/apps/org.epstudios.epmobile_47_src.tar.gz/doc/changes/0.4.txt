EP Mobile v0.4 Release Notes
============================

Changes since v0.3.1
--------------------

* Added Wide Complex Tachycardia Algorithms (Brugada, RWPT, Morphology).

* Added images to illustrate ECG morphologies.

* QTc Calculator now uses Heart Rate or RR Interval.
  
* Some general "prettifying" of layouts.

* Fixed wrong icon sizes.
