WikipOff

WikipOff (Wikipedia Offline) is an offline Wikipedia browser for Android Platforms.

Articles are being extracted from Wikipedia XML dumps, converted from WikiCode to HTML, compressed into LZMA and stored in indexed SQLite tables with FTS.

The tools required to build your own databases to be opened with WikipOff are available on this GitHub repository

The generated .sqlite files can be stored in the App storage directory. This may depend on your settings and Android version. Some examples:

  • /mnt/sdcard/fr.renzo.wikipoff/databases/ for a pre-4.4.2 android with a SD card storage.
  • /storage/sdcard/Android/data/fr.renzo.wikipoff/files/fr.renzo.wikipoff/databases for post-4.4.2 android with a SD card storage.
The correct folder to use is displayed on the storage settings view, just add fr.renzo.wikipoff/databases after the path shown there.

Direct links for generated files are available in :

  • Web : available_wikis.xml
  • On your device : <storage>/fr.renzo.wikipoff/available_wikis.xml
You can edit this last file to add your own hosted databases!
