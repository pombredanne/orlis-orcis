# Application:
OI ショッピングリスト

# Promo text:
OI ショッピングリスト でショッピングアイテムを記録することができます。

# Description:
OI ショッピングリストはチェックリストの作成・管理を簡単にする強力なアプリケーションです。 OI ショッピングリストはアイテムをリストに追加、チェックやリストから削除するのを簡単にします。 このアプリケーションは使用する各店の別々の値段をトラックすることができ、高度なカスタマイズができます。 フォントサイズ、並べ替え順序、リスト列やリストクリーンアップ動作はすべてユーザーがコントロールできる設定です。 これらの機能はすべて鮮明で様々なテーマでカスタマイズ可能なレイアウトで集結されています。 

この無料でオープンソースのアプリケーションのソースコードは以下の場所から取得できます： 
http://code.google.com/p/openintents

このアプリケーションは広告を含まず、インターネット許可を必要としません。 

すべての変更リストや質問リストを見るには： 
http://www.openintents.org

あなたもこの言語の翻訳を Launchpad で改善することができます： 
https://translations.launchpad.net/openintents/trunk

機能： 
* リストを追加・変数・削除。
* アイテムをすばやく簡単に作成・削除・チェック。
* 様々なテーマ。
* カスタマイズ可能なフォントサイズと並べ替え順序。
* 各店の別々の値段を記録する。
* カスタマイズ可能なリスト列。

"OI ショッピングリスト" で利用可能な拡張子： 
* "OI ボイスノート" でノートに録音したメッセージを追加。
* "OI CSV 変換" でショッピングリストをインポート・エクスポート。
* "OI 情報" のライセンス情報を表示する。
* "OI バーコードプラグイン" でアイテムを入力・チェック。

"OI ショッピングリスト" と動作するアプリケーション： 
* "OI アップデート" でアップデート。
* "マイバックアッププロ" でリストをバックアップ。
* "Bites"でレシピの材料をインポート。
