# Application:
OI Shopping List

# Promo text:
Garder une trace de vos courses avec OI Shopping List.

# Description:
OI Shopping Liste est une application puissante qui le rend facile de créer et gérer des listes de contrôle OI Shopping Liste, il est facile d'ajouter des éléments à une liste, cochez l'élément off, et le retirer de la liste. L'application permet de suivre des prix distincts pour chaque magasin que vous utilisez, et dispose d'un haut niveau de personnalisation. Taille de la police, l'ordre de tri, colonnes de la liste, et le comportement de nettoyage de liste sont tous les paramètres contrôlés par l'utilisateur. Toutes ces caractéristiques sont réunies dans une mise en croustillante qui est personnalisable grâce à une variété de thèmes. 

Le code source de cette application libre et open source est disponible à l'adresse: 
http://code.google.com/p/openintents

Cette application est petite annonce gratuite et ne nécessite pas l'autorisation d'Internet. 

Pour la liste complète des changements et une liste de questions fréquemment posées, s'il vous plaît visitez: 
http://www.openintents.org

https://translations.launchpad.net/openintents/trunk

caractéristiques: 
* Ajouter, modifier et supprimer des listes.
* Créer, supprimer et vérifier les articles rapidement et facilement.
* Variété de thèmes.
* Taille des caractères et l'ordre de tri.
* Gardez une trace des prix distincts pour chaque magasin.
* Colonnes de la liste personnalisable.

Extensions disponibles pour "OI Shopping List ": 
* Ajouter un message vocal à un acte avec "OI Voice Notes ".
* Importer / exporter des listes de courses avec "OI Convert CSV ".
* Voir les informations de licence avec "OI propos ".
* Entrez / vérifiez les éléments avec "Barcode OI Plugin ".

Applications qui fonctionnent avec "OI Shopping List ": 
* Mise à jour avec " OI Update".
* Listes de sauvegarde avec  "Mon Backup Pro ".
* Importer ingrédients de recettes avec "Morsures ".
