# Application:
OI Lista zakupów

# Promo text:
Zarządzaj swoimi listami zakupów używając OI Lista zakupów.

# Description:
OI Lista zakupów to potężna aplikacja, która umożliwia łatwe tworzenie i zarządzanie listami kontrolnymi. OI Lista zakupów ułatwia dodawanie elementów do listy, zaznaczanie i usuwanie ich z listy. Aplikacja może śledzić oddzielne cen dla każdego sklepu, którego używasz, i oferuje wysoki poziom personalizacji. Rozmiar czcionki, porządek sortowania, kolumny list, i zachowanie podczas czyszczenia list to ustawienia kontrolowane przez użytkownika. Wszystkie te funkcje są połączone w wyraźny wygląd, który można zmieniać za pomocą różnych motywów.

Cechy: 
* Dodawaj, edytuj i usuwaj listy.
* Twórz, usuwaj i sprawdzaj pozycje szybko i łatwo.
* Różnorodność motywów.
* Możliwość dostosowania rozmiaru czcionki oraz porządku sortowania.
* Śledź oddzielnie ceny dla każdego sklepu.
* Dostosuj kolumny listy.

Dostępne rozszerzenia dla "OI Lista zakupów": 
* Dodaj nagraną wiadomość do notatki używając "OI Voice Notes".
* Import/eksport list zakupów z "OI Konwertuj CSV".
* Pokaż informacje licencyjne za pomocą "OI About".
* Wprowadź/sprawdź pozycje używając "Barcode OI Plugin".

Aplikacje działające z "OI Lista zakupów": 
* Aktualizuj za pomocą "OI Uptade".
* Twórz kopie zapasowe list używając "My Backup Pro".
* Importuj składniki przepisów używając "Bites".
