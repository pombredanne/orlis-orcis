# Application:
Listă de cumpărături OI

# Promo text:
Țineți evidența articolelor de cumpărături cu Listă de cumpărături OI.

# Description:
Lista de cumpărături OI este o aplicaţie puternică și care simplifică gestionarea și crearea listelor de verificare. Lista de cumpărături OI simplifică adăugarea unui produs pe o listă, debifați produsul şi scoateţi-l din listă. Aplicația poate urmări separat preţuri pentru fiecare magazin pe care îl utilizaţi, şi dispune de un nivel ridicat de personalizare. Dimensiunea fontului, ordinea de sortare, coloanele listă, şi comportamentul listei de curățare sunt toate setările controlate de către utilizator. Toate aceste caracteristici sunt reunite într-un aspect proaspăt care este personalizabil printr-o varietate de teme. 

Codul sursă al acestei aplicații gratuite și cu sursă deschisă este disponibil la: 
http://code.google.com/p/openintents

Această aplicație nu conține reclame și nu necesită permisiunea de internet. 

Pentru lista completă cu modificări și o listă cu întrebări frecvente, vizitați: 
http://www.openintents.org

Puteți ajuta la îmbunătățirea traducerii în limba dumneavoastră la Launchpad: 
https://translations.launchpad.net/openintents/trunk

Caracteristici: 
* Adaugă, editează și șterge liste.
* Creează, șterge, și verifică articolele repede și ușor.
* Varietate de teme.
* Mărimea fontului și ordinul de sortare personalizabile.
* Țineți separat evidența prețurilor pentru fiecare magazin.
* Listă de coloane personalizabilă.

Extensii disponibile pentru „OI Shopping List”: 
* Adaugă un mesaj înregistrat unei note cu „OI Voice Notes”.
* Importă/exportă listă de cumpărături cu „OI Convert CSV”.
* Arată informaţiile de licenţă cu „OI About”.
* Introduceți/Bifați articole cu „Barcode OI Plugin”.

Aplicații care lucrează cu „OI Shopping List”: 
* Actualizaţi cu „OI Update”.
* Listă de rezervă cu „My Backup Pro”.
* Importă ingrediente ale rețetelor cu „Bites”.
