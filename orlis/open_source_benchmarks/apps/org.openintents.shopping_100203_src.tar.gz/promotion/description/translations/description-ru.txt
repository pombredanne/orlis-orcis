# Application:
OI Shopping List

# Promo text:
Keep track of your shopping items with OI Shopping List.

# Description:
OI Shopping List это мощное приложение позволяющее легко создавать и управлять списками. OI Shopping List позволяет легко добавлять элементы в список, отмечать их выполненными, и удалять их из списка. Приложение может отслеживать расценки для каждого магазина, который вы используете, и имеет высокий уровень настройки. Размер шрифта, порядок сортировки списка столбцов, и порядок очистки списка - это все параметры контролируемые пользователем. Все эти функции объединяются в четкий интерфейс, который настраивается с помощью различных тем. 

Исходный код этой программы является свободным и открытым. Доступен по адресу: 
http://code.google.com/p/openintents

Это приложение является бесплатным и не требует постоянного подключения к Интернет. 

С полным списком изменений и списком часто задаваемых вопросов можно ознакомиться на: 
http://www.openintents.org

Вы можете помочь улучшить перевод прилоежния на ваш язык, посетите Launchpad: 
https://translations.launchpad.net/openintents/trunk

Возможности: 
* Добавление, изменение и удаление списков.
* Создание, удаление и отметка элементов быстро и легко.
* Множество тем
* Настраиваемый размер шрифта и порядок сортировки.
* Следите за отдельными ценами для каждого магазина.
* Настраиваемые столбцы списка.

Доступные дополнения для "OI Shopping List": 
* Добавить записанное сообщение к заметке с помощью "OI Voice Notes".
* Импорт / экспорт списков покупок с помощью "OI Convert CSV".
* Показать информацию о лицензии с помощью "OI About".
* Ввод / проверка элементов с помощью "Barcode OI Plugin".

Приложения работающие с "OI Shopping List": 
* Обновить с помощью "OI Update".
* Резервное копирование списков с помощью "My Backup Pro".
* Импорт ингредиентов рецептов с помощью "Bites".
