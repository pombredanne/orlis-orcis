==============
UNREAL TRACKER
==============

Author: Sevag Hanssian <sevag.hanssian@gmail.com>

===========
DESCRIPTION
===========

This is intended to be an app to track Unreal Tournament and Unreal Engine 4 (tracking git commits, blog posts, marketplace listings, etc.).

========
FEATURES
========

* UT & UE blog heading aggregation
* New UE marketplace listings
* UT live stream countdown

=======
INSTALL
=======

The repository is an Intellij IDEA Android project.
